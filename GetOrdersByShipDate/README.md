# GetOrdersByShipDate (経理・売上処理自動化プロジェクト)

## 概要
ネクストエンジンから出荷済みとなった伝票情報をAPIで取得し、経理の売上計上処理に必要なデータ基盤を構築します。

## 開発背景と目的
従来、経理処理のためにネクストエンジンの管理画面から手動でCSVをダウンロードしていましたが、本プロジェクトによりその工程を完全自動化する見込みが立ちました。
「人の手を介さない自動化」は、私が開発において最も重要視しているテーマの一つです。

## 主な機能・特徴
- **経理業務の自動化**: 定期的に決まった形式でデータを抽出することで、毎月の売上処理の工数を大幅に削減。
- **ヒューマンエラーの排除**: 手動ダウンロードや加工によるミスを無くし、常に正確な「出荷済みデータ」を担保。
- **データ提供の定時性**: スケジュール実行（トリガー）を活用し、必要なタイミングで必要なフォーマットのデータを用意。

## 実務上のメリット
CSVの書き出し作業という「ルーチンワーク」をシステムに置き換えることで、人間がより付加価値の高い分析業務に集中できる環境を実現しています。
