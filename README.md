# NextEngine API Integration Suite (GAS)

ネクストエンジンAPIを活用し、EC運営における在庫・受注・売上管理を自動化するためのGoogle Apps Script（GAS）プロジェクト群です。

## 🚀 Key Achievement: 認証基盤の完全内製化と高速処理の実装
本プロジェクトの最大の実績は、**ネクストエンジン独自のOAuth2.0認証フローを完全に独自実装し、共通ライブラリ化した点**にあります。

* **認証の突破とライブラリ化**: 複雑な`uid`パラメータを含む認証フローを解析し、`認証_ライブラリ.gs`として共通化。2週間の試行錯誤を経て、AIとの「バイブコーディング」により突破。
* **高速データ処理**: 1回のAPIコールで1,000行、合計3,000行以上の在庫情報を**約18秒**で取得・同期するロジックを構築。
* **拡張性**: `00_認証ライブラリ使用必須関数.gs`を介して、新しいプロジェクトへ即座に認証機能を展開できるアーキテクチャを採用。

## 📂 Project Structure
現在、以下の7つの専門プロジェクトを運用しており、各業務を自動化しています。

1.  **GetInventoryData**: 在庫情報の高速取得・同期
2.  **GetOrdersByShipDate**: 出荷予定日に基づく受注データの抽出
3.  **GetSalesSummaryAcquisition**: 過去7日間の売上集計とトレンド分析
4.  **GetScheduledShipmentQuantity**: 将来の出荷予定数の可視化
5.  **GetSetMasterData**: 複雑なセット商品マスタの展開・取得
6.  **GetShipmentCompletionSlip**: 出荷済み伝票（同梱処理含む）のデータ管理
7.  **GetStoreMasterAcquisition**: 各モール・カートのマスタ情報一括管理
