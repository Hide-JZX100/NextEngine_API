# GetShipmentCompletionSlip (同梱伝票管理・経理連携自動化)

## 概要
ネクストエンジンから「出荷完了」ステータスの伝票データを取得し、特に複雑な「同梱処理」が施された伝票の紐付け情報を自動抽出します。

## 開発背景と目的
EC運営において、運賃コスト削減のための「同梱処理」は不可欠ですが、従来のCSV出力では同梱元・先の紐付け情報が不足していました。そのため、現場では伝票番号を手入力でメモし、経理へ連携するというアナログな高負荷作業が発生していました。
本プロジェクトは、APIを利用することで「手動では取得困難なデータ」を自動抽出し、コスト削減と業務効率化の両立を実現するために立ち上げられました。

## 主な機能・特徴
- **API限定データの活用**: CSVダウンロードでは取得できない「同梱元伝票番号」「同梱先伝票番号」を直接取得し、正確なデータ連携を実現。
- **フルクラウドによる自動化**: 物理サーバーを介さず、GASとAPIのみで完結するクリーンな自動処理フローを構築。
- **複雑なデータ構造の解析**: 同梱によって親子関係となった伝票情報を適切に整理し、経理処理が可能な形式へ変換。

## 実務上のメリット
本格運用に向けて開発を進めており、導入後は「運賃軽減のための同梱作業」に伴う事務コストを大幅に削減できる見込みです。現場の作業ミスを物理的に防ぐだけでなく、経理セクションの突合作業を自動化することで、全社的な生産性向上に寄与します。
